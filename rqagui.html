RQAGUI

About:

The rqaGUI provides a Graphical Frontend (GUI) to the Commandline Recurrence Plot Binary. It is a thin Python wrapper to ease the interaction with the commandline tool that allows for interactive setting of the parameters and a visualisation of RPs and RQA measures.

Contents:

rqaGui.py
- the frontend

Requirements:

  • the Commandline Recurrence Plot Binary (obviously)
  • either
    • Python — I currently run 2.6, but it was developed on 2.4, so this should be ok.
    • Tkinter — Python's de-facto standard GUI, should come w/ your distribution
    • matplotlib Required only if you want to plot things, not needed for data crunching.
    or
    • An all-in-one python distribution like the Enthought Python Distribution (available as a EPD Free license or EPD academic license)

Known Issues:

  • on windows there are occasionally problems with piping to the binary, let me know if you have trouble

Instructions:

Setup Python w/ TKinter and matplotlib

Unix/Linux/OSX

  • have the rp binary (renamed to 'rp') in your $PATH and executable
    # add path (in Bash and the likes)
export PATH=$PATH:/path/to/rp
# add path (in CSH)
setenv PATH "${PATH}:/path/to/rp 
#make executable
chmod u+x rp
  • Save rqaGUI.py in your $PATH with rights set to executable 
    chmod u+x rqaGUI.py

Windows*

  • save the binary as 'rp.exe' in the same directory/folder as the rpaGUI.py file
  • Setup Python w/ TKinter and matplotlib
  • start rqaGUI.py by double-clicking it

* tested on Win7 64bit in Virtualbox
